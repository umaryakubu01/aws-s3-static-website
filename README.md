# aws-s3-static-website
Simple HTML and CSS static website hosted on AWS S3
## Features
- Simple and clean UI
- Fully Static Website
- Hosted on AWS S3

## Technologies Used
- HTML
- CSS
- AWS S3

## Project Goal
To learn how to:
- Create a static website
- Use Github for version control
- Deploy a website using AWS S3
